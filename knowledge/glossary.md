# 用語集（Glossary）

論文要約・プログラム雛形・ルールエンジン内で**統一して使用する用語**を定義する。表記ゆれを避けるための単一の参照源。

## 強度・負荷

| 用語 | 定義 | 補足 |
| --- | --- | --- |
| 1RM | One Repetition Maximum / 1 回挙上できる最大重量 | 推定値で運用してよい |
| %1RM | 1RM に対する割合（パーセント） | メインセット強度の指標 |
| RPE | Rate of Perceived Exertion / 主観的運動強度 | 1〜10 スケール。RPE 8 = 「あと 2 レップ余力あり」 |
| RIR | Reps in Reserve / 余力レップ数 | RIR 2 ≒ RPE 8 |

## ボリューム

| 用語 | 定義 |
| --- | --- |
| セット数 | 1 種目あたりの実施セット数 |
| 週間ボリューム | 1 週間あたりの筋群別総セット数 |
| MEV | Minimum Effective Volume / 最低有効ボリューム |
| MAV | Maximum Adaptive Volume / 最大適応ボリューム |
| MRV | Maximum Recoverable Volume / 最大回復可能ボリューム |

## 種目分類

| 用語 | 定義 |
| --- | --- |
| コンパウンド種目 | 多関節種目。BIG3、ロウ系、プレス系 |
| アイソレーション種目 | 単関節種目。カール、エクステンション系 |
| BIG3 | ベンチプレス、スクワット、デッドリフトの総称 |

## プログラム

| 用語 | 定義 |
| --- | --- |
| Full Body | 1 セッションで全身を扱うスプリット |
| Upper-Lower | 上半身／下半身分割 |
| PPL | Push / Pull / Legs の 3 分割 |
| 周期化 | ボリュームと強度を段階的に変化させる手法 |
| デロード | 計画的に重量・ボリュームを下げる回復週 |

## 安全・医療

| 用語 | 定義 |
| --- | --- |
| safety_flags | サーバーが返す安全フラグ。`pain_reported`, `session_suspended`, `partial_skip` 等 |
| advisory.level | 推奨アクションの強度。`none`, `partial_skip`, `rest_or_consult`, `deload` |

> 本アプリは医療助言を提供しない。痛み・違和感がある場合は運動を中止し、医療専門家へ相談すること。
