# AI知識基盤 — プログラムテンプレート定義書

## 適用ルール
AIはこのファイルに定義されたテンプレートから選択・調整するのみ。
テンプレート外のゼロからの生成は禁止。
テンプレートの追加・変更はCPOの承認が必要。

---

## テンプレート選択ロジック

```
入力: 習熟度 + 週の利用可能日数 + 目的

選択:
  初心者 + 週2〜3日 → TMPL-01（全身法・週3）
  中級者 + 週3〜4日 → TMPL-02（上下分割・週4）
  中級者 + 週4〜6日 → TMPL-03（PPL分割・週6）

不明な入力:
  週日数が不明 → TMPL-01を選択（最も安全なデフォルト）
  習熟度が不明 → 初心者として扱う
```

---

## TMPL-01: 初心者テンプレート（全身法・週3）

**対象:** 初心者 / 週2〜3日利用可能
**特徴:** 全身を1回のセッションで鍛える。回復時間が十分取れる。

### セッション構成（A/B交互）

**Session A:**
| 種目 | セット数 | レップ数 | 休憩 |
|------|---------|---------|------|
| スクワット（またはレッグプレス） | 3 | 8〜10 | 90秒 |
| ベンチプレス（またはダンベルプレス） | 3 | 8〜10 | 90秒 |
| ベントオーバーロウ（またはシーテッドロウ） | 3 | 8〜10 | 90秒 |
| ショルダープレス | 2 | 10〜12 | 60秒 |
| プランク | 2 | 30秒保持 | 60秒 |

**Session B:**
| 種目 | セット数 | レップ数 | 休憩 |
|------|---------|---------|------|
| デッドリフト（またはルーマニアンデッドリフト） | 3 | 6〜8 | 120秒 |
| インクラインプレス（またはプッシュアップ） | 3 | 8〜10 | 90秒 |
| ラットプルダウン（またはチンアップ） | 3 | 8〜10 | 90秒 |
| ダンベルカール | 2 | 10〜12 | 60秒 |
| トライセプスプレスダウン | 2 | 10〜12 | 60秒 |

**週スケジュール例:**
- 月: Session A / 水: Session B / 金: Session A
- 次週: 月: Session B / 水: Session A / 金: Session B（交互）

**調整ルール（AIが行う微調整）:**
- 記録済み重量がある種目は `前回重量 + 2.5kg` を提案（全セット達成済みの場合のみ）
- 記録なしの場合は軽めの重量から開始するよう案内（具体的な重量は提案しない）

## TMPL-02: 中級者テンプレート（上下分割・週4）

**対象:** 中級者 / 週3〜4日利用可能
**特徴:** 上半身・下半身を交互に鍛える。各部位の回復を確保しながら頻度を上げる。

### セッション構成

**Upper（上半身）:**
| 種目 | セット数 | レップ数 | 休憩 |
|------|---------|---------|------|
| ベンチプレス | 4 | 6〜10 | 90秒 |
| ベントオーバーロウ | 4 | 6〜10 | 90秒 |
| ショルダープレス | 3 | 8〜12 | 90秒 |
| ラットプルダウン | 3 | 8〜12 | 90秒 |
| ダンベルカール | 3 | 10〜12 | 60秒 |
| トライセプスエクステンション | 3 | 10〜12 | 60秒 |

**Lower（下半身）:**
| 種目 | セット数 | レップ数 | 休憩 |
|------|---------|---------|------|
| スクワット | 4 | 6〜10 | 120秒 |
| ルーマニアンデッドリフト | 4 | 8〜10 | 120秒 |
| レッグプレス | 3 | 10〜12 | 90秒 |
| レッグカール | 3 | 10〜12 | 90秒 |
| カーフレイズ | 3 | 12〜15 | 60秒 |

**週スケジュール例:**
- 月: Upper / 火: Lower / 木: Upper / 金: Lower

**調整ルール:**
- TMPL-01と同様（前回重量 + 2.5kg上半身 / + 5kg下半身）

## TMPL-03: 中級者テンプレート（PPL分割・週6）

**対象:** 中級者 / 週4〜6日利用可能
**特徴:** Push（押す）/ Pull（引く）/ Legs（脚）の3分割を週2サイクル行う。

### セッション構成

**Push（胸・肩・三頭筋）:**
| 種目 | セット数 | レップ数 | 休憩 |
|------|---------|---------|------|
| ベンチプレス | 4 | 6〜8 | 90秒 |
| インクラインダンベルプレス | 3 | 8〜12 | 90秒 |
| ショルダープレス | 3 | 8〜12 | 90秒 |
| サイドレイズ | 3 | 12〜15 | 60秒 |
| トライセプスプレスダウン | 3 | 10〜12 | 60秒 |

**Pull（背中・二頭筋）:**
| 種目 | セット数 | レップ数 | 休憩 |
|------|---------|---------|------|
| デッドリフト | 3 | 5〜6 | 180秒 |
| ラットプルダウン | 4 | 8〜12 | 90秒 |
| シーテッドロウ | 4 | 8〜12 | 90秒 |
| フェイスプル | 3 | 12〜15 | 60秒 |
| ダンベルカール | 3 | 10〜12 | 60秒 |

**Legs（大腿四頭筋・ハムストリング・臀部・ふくらはぎ）:**
| 種目 | セット数 | レップ数 | 休憩 |
|------|---------|---------|------|
| スクワット | 4 | 6〜10 | 120秒 |
| レッグプレス | 4 | 10〜12 | 90秒 |
| レッグカール | 3 | 10〜12 | 90秒 |
| ブルガリアンスプリットスクワット | 3 | 10〜12（片側） | 90秒 |
| カーフレイズ | 4 | 12〜15 | 60秒 |

**週スケジュール例:**
- 月: Push / 火: Pull / 水: Legs / 木: Push / 金: Pull / 土: Legs / 日: 休息

## テンプレート使用時の必須出力フォーマット

AIがテンプレートを提案する際は、以下の形式で出力する：

```
【使用テンプレート】TMPL-XX（テンプレート名）
【選択理由】週N日 / 習熟度: 初心者/中級者

【本日のメニュー】
種目: ○○
セット数: N セット
レップ数: N〜N 回
目安重量: [前回記録がある場合は前回重量+2.5kg / ない場合は「軽めから開始してください」]

【注意】
一般的なトレーニング理論に基づく提案です。
```

## 提案禁止ケース

以下の場合はテンプレート提案を行わず、代替メッセージを返す：

| ケース | 返すメッセージ |
|--------|-------------|
| 週1日以下の利用予定 | 「週2日以上のトレーニングを確保できる場合にメニューを提案します。」 |
| 上級者と自己申告 | 「上級者向けの個別プログラムはサポート対象外です。現在のプログラムを継続してください。」 |
| 怪我・痛みの言及あり | safety_constraints.mdの対応フロー参照 |
