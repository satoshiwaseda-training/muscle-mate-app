# tactics — 分割法

## 定義
分割法とは、1週間のトレーニングをどの部位の組み合わせで行うかの設計。
週の利用可能日数と習熟度によって最適な分割が変わる。

---

## 全身法（Full Body）

**対象:** 初心者 / 週2〜3日
**原理:** 毎セッションで全身の主要部位を刺激する。頻度が高く、初心者の神経適応に最適。

**特徴:**
- 1部位あたりの週頻度: 2〜3回
- 1セッションあたりのボリューム: 少ない（6〜9セット/部位/週）
- 回復時間: セッション間48時間確保が必須

**週スケジュール例（週3日）:**
```
月: 全身A / 水: 全身B / 金: 全身A（翌週は全身Bから開始）
```

**セッション内の種目数目安:** 4〜6種目（主要コンパウンド種目 + 1〜2のアイソレーション）

---

## 上下分割（Upper / Lower）

**対象:** 初心者後期〜中級者 / 週3〜4日
**原理:** 上半身と下半身を交互に鍛える。1部位あたりの週頻度2回を維持しながらボリュームを増やせる。

**特徴:**
- 1部位あたりの週頻度: 2回
- 1セッションあたりのボリューム: 中程度（8〜12セット/部位/週）
- 回復時間: 上半身→下半身は翌日可。同一部位は48時間以上

**週スケジュール例（週4日）:**
```
月: 上半身 / 火: 下半身 / 木: 上半身 / 金: 下半身
```

**週3日の場合:**
```
月: 上半身 / 水: 下半身 / 金: 上半身（翌週は月: 下半身から開始）
```

---

## PPL分割（Push / Pull / Legs）

**対象:** 中級者 / 週4〜6日
**原理:** 動作パターン別に分割。同じ関節・筋群を使う種目を同日に集中させる。

**Push（胸・肩・三頭筋）:**
胸・肩・三頭筋を使う「押す動作」の種目を集める。

**Pull（背中・二頭筋）:**
背中・二頭筋を使う「引く動作」の種目を集める。

**Legs（大腿四頭筋・ハムストリング・臀部・ふくらはぎ）:**
脚全体の種目を集める。

**週スケジュール例（週6日）:**
```
月: Push / 火: Pull / 水: Legs / 木: Push / 金: Pull / 土: Legs / 日: 休息
```

**週4日（変形PPL）:**
```
月: Push / 火: Pull / 木: Legs / 金: Push or Full Body
```

---

## 分割法の選択ロジック（AIが使用）

```
週2〜3日 → 全身法
週3〜4日 → 上下分割（初心者：3日全身法も可）
週4〜6日 → PPL分割（中級者のみ）
週1日以下 → プログラム提案不可（week 2日以上を推奨）
```
