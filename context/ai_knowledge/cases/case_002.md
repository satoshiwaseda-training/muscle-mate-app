# cases — CASE-002: 初心者・週3日・重量が3週間増えていない（ストール）

## 1. 状態（入力情報）
- 習熟度: 初心者
- 週利用可能日数: 3日
- 目的: 筋肥大
- 記録状況: 3週間分の記録あり。ベンチプレス60kgで3週間停滞。毎回3セット中、最終セットで設定回数（8〜10回）の下限（8回）に届いていない。

---

## 2. 評価（問題点の特定）

**参照: principles/progressive_overload.md, principles/recovery_theory.md**

現状の分析:
- 3週間同一重量 → ストール（停滞）と判定
- 最終セットで下限未達 → 全セット達成していないため重量増加の条件を満たしていない
- 考えられる原因:
  A. ボリュームが少なすぎる（週6〜8セットがMEVを下回っている可能性）
  B. 回復不足（睡眠・栄養は確認できないが、疲労蓄積の可能性）
  C. 重量が適切でない（RPEが高すぎる設定になっている）

---

## 3. 改善方針

**参照: principles/progressive_overload.md（ストール対応Step 1〜2）**

方針: 重量を下げず、回数を増やすことでProgressを再開させる。

---

## 4. 実践プラン

```
【評価】
ベンチプレス60kgで3週間停滞しています。
最終セットで設定回数の下限（8回）に届いていないことを確認しました。

【改善方針】
重量は据え置き、回数の上限を引き上げることでProgressを再開させます。

【実践プラン（次回セッション）】
ベンチプレス:
  - 重量: 60kg（変更なし）
  - 目標回数: 8〜12回（上限を10回→12回に引き上げ）
  - セット数: 3セット（変更なし）
  - 目標: 3セット全てで12回達成できたら次回+2.5kg

その他の種目: 前回通りに継続

【理由】
全セットで設定回数の上限を達成していないため、重量増加の条件を満たしていません。
まず回数を伸ばすことで「達成」の状態を作り、次回の重量増加につなげます。
これはProgressive Overloadの「回数増加」バリアントです。

一般的なトレーニング理論に基づく提案です。
```

---

## 5. このケースから得られる知見

- ストール時の最初の対応は「重量を変えず回数を伸ばす」
- 「3週間同じ重量」は停滞ではなく、回数が伸びていればProgressしている
- 重量を下げる判断は最終手段（Step 3）
