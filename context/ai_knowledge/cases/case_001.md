# cases — CASE-001: 初心者・週3日・停滞なし

## 1. 状態（入力情報）
- 習熟度: 初心者
- 週利用可能日数: 3日
- 目的: 筋肥大
- 記録状況: 2週間分の記録あり。毎回全セット達成。

---

## 2. 評価（問題点の特定）

**参照: principles/progressive_overload.md**

現状の分析:
- 全セット達成が継続している → Progressが順調に進んでいる
- 2週間継続 → 習慣化の初期段階。変更よりも継続を優先すべき
- 問題点: なし。理想的な状態。

---

## 3. 改善方針

**参照: principles/progressive_overload.md, volume_theory.md**

方針: 重量を増加させてProgressを維持する。ボリュームはMEV〜MAVの範囲で現状維持。

---

## 4. 実践プラン

**参照: tactics/split_methods.md（全身法）, tactics/exercise_selection.md**

```
【評価】
2週間の記録から、全セット・全種目で設定回数を達成しています。
現在のプログラムは順調に進行しています。

【改善方針】
重量を増加させてProgressを継続します。

【実践プラン（次回セッション）】
- スクワット: 前回50kg → 今回55kg（+5kg）
- ベンチプレス: 前回40kg → 今回42.5kg（+2.5kg）
- ラットプルダウン: 前回45kg → 今回47.5kg（+2.5kg）
- その他の種目: 同重量で継続

【理由】
全セットで設定回数の上限（10回）を達成したため、
Progressive Overloadの原則に基づき重量を増加します。
下半身（スクワット）は+5kg、上半身は+2.5kgが標準的な増加量です。

一般的なトレーニング理論に基づく提案です。
```

---

## 5. このケースから得られる知見

- 初心者は毎セッションで重量を増やせる可能性がある（神経適応が急速）
- 「うまくいっているプログラムは変えない」が基本原則
- ボリューム増加よりも重量増加を優先する段階
