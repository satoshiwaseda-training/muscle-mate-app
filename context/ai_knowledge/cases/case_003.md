# cases — CASE-003: 中級者・週4日・ボリューム不足の診断

## 1. 状態（入力情報）
- 習熟度: 中級者（トレーニング歴1年）
- 週利用可能日数: 4日
- 目的: 筋肥大
- 記録状況: 上下分割で4ヶ月実施。胸のベンチプレスは4セット×3日/週 → 週12セット。背中は3セット×2日/週 → 週6セット。重量は増えているが背中の発達が遅れている。

---

## 2. 評価（問題点の特定）

**参照: principles/volume_theory.md**

現状の分析:
- 胸: 週12セット → MAV範囲（12〜18セット）の下限。適切。
- 背中: 週6セット → MEV（8〜10セット）を下回っている。**ボリューム不足と診断。**
- 背中の発達が遅れている → ボリューム不足が原因と推定される
- 重量が増えているため、強度（MEV条件）は問題なし

---

## 3. 改善方針

**参照: principles/volume_theory.md（MEV〜MAV調整）**

方針: 背中のセット数を週8〜10セットに増やしてMEVに到達させる。
一度に増やしすぎず、週+2セットから開始する（急激なボリューム増加は回復負荷増大のため）。

---

## 4. 実践プラン

```
【評価】
背中のトレーニングボリュームが週6セットと、一般的に有効とされる最低限（週8〜10セット）を下回っています。
背中の発達が遅れている原因として、ボリューム不足が最も可能性が高いと判断します。

【改善方針】
背中のセット数を週6セット → 8セットに増やします（週+2セット）。
4週間継続後、発達状況を確認してさらに増やすかを判断します。

【実践プラン（上半身セッション）】
現行 → 修正後:
- ラットプルダウン: 3セット → 4セット（+1セット）
- ベントオーバーロウ: 3セット → 4セット（+1セット）
合計: 週6セット → 週8セット

胸・肩・腕の種目: 変更なし

【理由】
ボリューム理論では、筋肥大に必要な最低ボリューム（MEV）は中級者で週8〜10セット/部位です。
週6セットはこれを下回っているため、まずMEVに到達させることを優先します。
一度に大幅に増やすと回復が追いつかなくなるため、週+2セットを上限としています。

一般的なトレーニング理論に基づく提案です。
```

---

## 5. このケースから得られる知見

- 「重量が増えているのに発達しない」はボリューム不足のサインである可能性が高い
- 部位別にボリュームを診断することで、偏りを発見できる
- ボリューム増加は週+2セットを上限とし、急激な増加を避ける
