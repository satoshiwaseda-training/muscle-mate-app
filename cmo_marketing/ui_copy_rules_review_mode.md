# CMO成果物 — UI文言ルール（審査通過モード）

## 適用範囲
アプリ内の全テキスト（ボタン・ラベル・説明文・エラーメッセージ・空状態テキスト）に適用。

---

## 禁止ワード（全文言で使用禁止）

| 禁止ワード | 理由 | 代替表現 |
|----------|------|---------|
| AI | App Store審査でのAI機能への厳しい審査を避ける | 「提案」「入力内容に基づく」 |
| 最強 | 誇大表現 | 使わない |
| 革命 | 誇大表現 | 使わない |
| 自動最適化 | AI想起 + 誇大表現 | 「入力に基づいて調整」 |
| スマート（AI想起） | AI想起 | 「シンプルな」「使いやすい」 |
| パーソナライズ | AI・データ収集の印象 | 「入力内容に応じた」 |
| あなたに最適な | 誇大表現 | 「一般的な理論に基づいた」 |

---

## 必須フレーズ（AI提案画面に必ず含める）

```
「一般的なトレーニング理論に基づく提案です」
「入力された情報はトレーニング提案の生成に使用されます」
```

---

## 文言変換対照表

### AI機能関連

| NG文言 | OK文言 |
|--------|--------|
| 「最適なトレーニングをAIが生成」 | 「入力内容に基づいてトレーニング案を提案します」 |
| 「AIがあなたのプログラムを最適化」 | 「記録に基づいてメニューを調整します」 |
| 「スマートなトレーニング提案」 | 「トレーニング提案」 |
| 「AIコーチ」 | 「トレーニング提案機能」 |
| 「自動でプログラムを生成」 | 「テンプレートからメニューを提案します」 |

### 機能説明

| NG文言 | OK文言 |
|--------|--------|
| 「Coming Soon」 | 使用禁止（機能を実装してから表示する） |
| 「近日公開予定」 | 使用禁止 |
| 「準備中」 | 使用禁止（該当UIを非表示にする） |
| 「ベータ版」 | 使用禁止 |

### エラーメッセージ

| NG文言 | OK文言 |
|--------|--------|
| 「エラーが発生しました」（のみ） | 「保存できませんでした。もう一度お試しください。」 |
| 「不明なエラー」 | 「処理を完了できませんでした。アプリを再起動してお試しください。」 |
| 空のエラー表示 | 必ず説明テキストを表示する |

### 空状態テキスト（必須）

| 画面 | 空状態テキスト（固定） |
|------|-------------------|
| ホーム（今日の記録なし） | 「まだ記録がありません」「＋ボタンから記録を追加してください」 |
| 履歴（記録なし） | 「記録がありません」「ホームタブから記録を追加してください」 |
| ベスト（記録なし） | 「まだ記録がありません」「ホームタブで記録を追加すると、自動的にベストが表示されます」 |
| 提案（入力不足） | 「トレーニング頻度と経験を入力すると、メニューを提案します」 |

---

## 外部通信の説明文（必須・変更禁止）

外部通信が発生する場合、UIに以下のいずれかを表示する：

**初回または通信前の説明（設定画面または提案画面）:**
```
「入力された情報はトレーニング提案の生成に使用されます。
氏名・メールアドレスなどの個人情報は送信されません。」
```

**通信が発生しない場合（ローカルのみ）:**
```
「すべてのデータはお使いの端末に保存されます。
外部サーバーへの送信は行いません。」
```

## チェックリスト（実装前にCMOが確認）

- [ ] 禁止ワードが全テキストに含まれていないか
- [ ] 空状態のある全画面に説明テキストが設定されているか
- [ ] AI提案画面に必須フレーズが含まれているか
- [ ] Coming Soon / 準備中 / 近日公開 が使われていないか
- [ ] 実装されていない機能の説明がないか
- [ ] エラーメッセージが空でないか（全エラーケースに文言があるか）
- [ ] 外部通信がある場合、送信内容の説明文が表示されているか
