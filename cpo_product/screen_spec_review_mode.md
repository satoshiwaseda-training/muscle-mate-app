# CPO成果物 — 審査通過モード 画面仕様書

## 前提
- 4画面固定。追加禁止。
- 全ボタンはタップ時に動作すること（押せないボタン禁止）
- 空状態には必ずテキストを表示すること
- Coming Soon / ダミーUI 禁止

---

## 画面構成図

```
┌─────────────────┐
│   BottomNavBar  │
│  記録  履歴  ベスト  設定 │
└─────────────────┘

画面1: 記録追加（HomeScreen）
画面2: 履歴（HistoryScreen）
画面3: ベスト（BestScreen）
画面4: 設定（SettingsScreen）
```

BottomNavigationBarで4画面を切り替える。
モーダル（記録入力フォーム）は画面1から呼び出す。

---

## 画面1: 記録追加（HomeScreen）

### 目的
今日のトレーニングを記録する。

### 画面要素

**ヘッダー**
- タイトル: 「今日の記録」
- 右上ボタン: 「＋」→ 記録入力モーダルを開く

**ボディ（今日の記録リスト）**
- 今日の記録が存在する場合:
  - 種目名 / 重量kg × 回数 × セット数 を1行で表示
  - 右端に「削除」ボタン（タップで確認ダイアログ → 削除）
- 今日の記録がゼロの場合:
  - 中央に「まだ記録がありません」のテキストを表示
  - その下に「＋ボタンから記録を追加してください」のサブテキストを表示

**記録入力モーダル（＋ボタンタップ時）**

| 入力項目 | 入力タイプ | バリデーション |
|---------|-----------|-------------|
| 種目名 | テキスト入力（キーボード） | 1文字以上・50文字以内 |
| 重量（kg） | 数値入力（小数点対応） | 0以上・999以下 |
| 回数 | 数値入力（整数） | 1以上・999以下 |
| セット数 | 数値入力（整数） | 1以上・99以下 |

- 保存ボタン: 全項目入力済みの場合のみ活性化（未入力時はグレーアウト＋タップ不可）
- キャンセルボタン: モーダルを閉じる（入力内容は破棄）
- バリデーションエラー時: 該当項目の下にエラーテキストを赤字で表示

## 画面2: 履歴（HistoryScreen）

### 目的
過去の全記録を日付ごとに確認する。

### 画面要素

**ヘッダー**
- タイトル: 「記録履歴」

**ボディ（日付グループ別リスト）**
- 記録が存在する場合:
  - 日付をセクションヘッダーとして表示（例: 「2026年4月5日（日）」）
  - その日の記録を縦に並べる（種目名 / 重量 × 回数 × セット数）
  - 新しい日付が上に来る降順ソート
- 記録がゼロの場合:
  - 中央に「記録がありません」のテキストを表示
  - その下に「ホームタブから記録を追加してください」のサブテキストを表示

**個別記録のスワイプ削除**
- 左スワイプで「削除」ボタンを表示
- タップで確認ダイアログ → 削除

---

## 画面3: ベスト（BestScreen）

### 目的
種目ごとの自己ベスト（最大重量）を確認する。

### 画面要素

**ヘッダー**
- タイトル: 「自己ベスト」

**ボディ（種目別ベストリスト）**
- 記録が存在する場合:
  - 種目名ごとにグループ化
  - 各種目の「最大重量 × その時の回数」を表示
  - 例: 「ベンチプレス — 100kg × 5回」
  - 種目名のアルファベット順 / 五十音順でソート
- 記録がゼロの場合:
  - 中央に「まだ記録がありません」のテキストを表示
  - その下に「ホームタブで記録を追加すると、自動的にベストが表示されます」のサブテキストを表示

---

## 画面4: 設定（SettingsScreen）

### 目的
アプリ情報・法的情報へのアクセスを提供する。

### 画面要素

**ヘッダー**
- タイトル: 「設定」

**セクション: データ管理**
| 項目 | 動作 |
|------|------|
| 全データを削除 | タップ → 確認ダイアログ（「削除すると元に戻せません」） → 削除実行 |

**セクション: アプリ情報**
| 項目 | 動作 |
|------|------|
| 利用規約 | タップ → 利用規約テキストを全画面で表示（WebViewではなく静的テキスト） |
| プライバシーポリシー | タップ → プライバシーポリシーテキストを全画面で表示（静的テキスト） |
| お問い合わせ | タップ → メールアプリを開く（宛先: support@muscle-mate.app） |
| バージョン | 静的テキストで表示（例: 「1.0.0」）タップ不可・グレー表示 |

**注意:**
- 利用規約・プライバシーポリシーはWebView / 外部URLを使わない（ネットワーク通信禁止のため）
- 必ずアプリ内の静的テキストとして実装する

## 画面遷移図

```
BottomNavBar
  ├── HomeScreen（記録追加）
  │     └── 記録入力モーダル（＋タップ）
  │           ├── 保存 → モーダルを閉じて一覧を更新
  │           └── キャンセル → モーダルを閉じる
  ├── HistoryScreen（履歴）
  │     └── 削除確認ダイアログ
  ├── BestScreen（ベスト）
  └── SettingsScreen（設定）
        ├── 全データ削除確認ダイアログ
        ├── 利用規約テキスト画面
        └── プライバシーポリシーテキスト画面
```
